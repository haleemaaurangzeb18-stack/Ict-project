# Ict-project
midnight pantry
1. Introduction:
Online shopping has become very common today. People prefer buying groceries online because it saves time and effort.
Midnight Pantry is an online grocery store website that allows users to view grocery items and shop easily at any time. This project was created to learn the basics of website development.
2. Objective:
To create a simple online grocery website
To display grocery items clearly
To make shopping easy for users
To learn basic web development skills
3. Technologies Used:
HTML – To create the structure of the website
CSS – To design and style the pages
JavaScript – To add basic functionality (if used)
4. Description of the Website:
The Midnight Pantry website shows different grocery products with their names, prices, and images.
Users can easily browse items and add them to the cart. The website is simple, clean, and easy to use.
5. Advantages:
Easy to use
Saves time
Available 24/7
Simple design for users
6. Limitations:
No online payment system
No user login system
Limited features because it is a beginner project
7. Conclusion:
The Midnight Pantry project helped me understand how websites work. I learned how to use HTML, CSS, and JavaScript to build a simple website. This project improved my skills
